Fantasy AGE

Published by Green Ronin, Fantasy AGE is a tabletop role-playing game (TTRPG) that is easy to learn, fun to play, and welcoming to new role players. It is powered by the popular Adventure Game Engine (AGE), which also powers other RPGs such as The Expanse, Modern AGE, Blue Rose, Cthulhu Awakens, and Fifth Season.


Welcome to the Fantasy AGE Character Creator!

This application allows you to easily create a character for the Fantasy AGE TTRPG. No math needed, and no flipping back and forth through the book, simply select "Create Character" below to start building a character!
